Task IDNOX-006
Feature slugdaily-prophet-deseason

Overview

Statistics verified on synthetic planted daily fixtures, deterministic, no network. Prophet runs in MAP mode on small series (fast); the prophet-dependent tests skip with a clear marker if prophet is unavailable, while the lazy-import + harmonic-fallback test always runs.

Acceptance tests

AT1Daily compositing + coverage + weekday counts
Covers
REQ-001 · REQ-002 · AC-001
Given
A synthetic per-overpass set (or daily cube) with planted cloud gaps.
Then
The daily footprint has NaN on gap days (no interpolation); the valid-day fraction and per-weekday valid counts are emitted.
AT2Prophet recovers components + activity residual
Covers
REQ-010 · REQ-012 · AC-002
Given
A planted daily series trend + yearly + weekly + activity + noise with cloud gaps.
Then
The fitted trend/yearly/weekly match the planted components within tolerance; the residual correlates with the planted activity; the residual is present only on observed days.
AT3Determinism
Covers
REQ-011 · NFR-005 · AC-003
Then
Two fits on identical input/config (MAP, uncertainty_samples=0) yield identical residuals.
AT4Weekday source-separation diagnostic
Covers
REQ-020 · REQ-021 · AC-004
Given
(a) a planted weekly cycle; (b) a flat-weekly series.
Then
(a) the weekday amplitude + variance-removed are recovered; (b) the weekly amplitude is ≈0 — the diagnostic does not invent a cycle.
AT5Comparison + catalyst re-run
Covers
REQ-030 · REQ-031 · AC-005
Then
The Prophet residual flows through the existing compare contract (full/regime/lead-lag vs BF, alongside yoy/STL/intensity/harmonic) and through the NOX-004 catalyst (a decomposition parquet it can consume).
AT6Causal refit — no look-ahead
Covers
REQ-032 · NFR-003 · AC-006
Then
The rolling-refit residual at day t is unchanged when future days are appended (the in-sample full-fit residual, by contrast, does change — the test asserts the causal path is used for any lead claim).
