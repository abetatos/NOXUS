Acceptance Tests — Steel emission-intensity (decoupling) model

Task IDNOX-003.1
Feature slugsteel-intensity-model
Statusspec_draft

Overview

All statistical behaviour is verified on synthetic planted fixtures (deterministic), never on live data. The headline guarantees are: (1) the decomposition recovers a planted trend and residual and reproduces the sign inversion, (2) smoothness is selected without the benchmark, and (3) no smoothing choice manufactures a correlation on a planted-null. Network is never touched.

Acceptance tests

AT101Planted decomposition → trend + residual + sign inversion
Covers
REQ-101 · AC-101
Given
A synthetic series signal = activity + s_true(t) + β·meteo + noise with a known smooth trend and a trend that dominates the variance.
When
fit_intensity_trend runs (meteo already removed).
Then
The fitted trend matches s_true within tolerance; residual correlates with activity; corr(levels, activity) < 0 while corr(residual, activity) > 0.
AT102CV selects smoothness without the benchmark
Covers
REQ-102 · AC-102 · NFR-102
Given
A planted trend of known smoothness.
When
Blocked time-series CV runs over the df grid.
Then
Selected df is near the planted smoothness; the selection path takes no benchmark argument (enforced by signature/test); selected df + criterion + cv_score are recorded.
AT103Sensitivity sweep + planted-null flatness
Covers
REQ-103 · AC-103
Given
(a) a planted-signal series; (b) an independent planted-null pair.
When
smoothness_sweep runs across the df grid.
Then
It returns one row per df (residual_r/p, peak_lag, levels_r) with the CV point identifiable; on the planted-null, residual_r stays near 0 across all df (no df manufactures a correlation).
AT104Dual-sign + decoupling report with CIs
Covers
REQ-104 · REQ-110 · REQ-131 · AC-104

A completed decomposition + validation run.

The report is generated.
Then
It emits the s(t) diagnostic series and states both levels_r and residual_r with p and CI, classifies vs the r≈0.5–0.75 bar, states the null when present, frames the negative levels term as decoupling, and emits no absolute tonnage.
