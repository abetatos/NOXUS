Technical summary

A new noxus/catalyst/ package, four testable stages behind small contracts, consuming the NOX-003.1 residual and the NOX-001 benchmark and ending in a catalyst report. The statistics are event-study, not time-series correlation: detect discrete NO₂ events, label production events, match them (precision/recall/lead), then study market abnormal returns around the events.

  1. Event detection (events.py): on the meteo-normalised, intensity-detrended residual, run a robust anomaly detector (median/MAD z) + a change-point/CUSUM detector; coverage-screen; strengthen the meteo control (ventilation index, stagnation flag) so a still-air spike is rejected. Emit a typed event table (date, direction, magnitude, detector).
  2. Ground truth (groundtruth.py): BF operating-rate jump events + a public curtailment calendar; combine + cause-tag.
  3. Event study vs production (study.py): match within a window; precision / recall / FPR / signed lead time with CIs; lead-vs-publication test.
  4. Market study (market.py ingest + study.py): daily prices for miners / steel ETF / (best-effort) ferrous futures; abnormal returns + CAR over [−k,+k] trading days; directional hit rate; production-confirmed vs unconfirmed split.

The catalyst report ties it together with strict no-look-ahead (causal baselines; trade at the next session after overpass+latency) and multiple-testing transparency.

Test design

  • Planted events: residual with known step surges/drops → detector recovers dates/directions; precision/recall correct (AT1).
  • Planted null: pure-noise residual → low FP rate (AT2).
  • Screens: cloud-gap + stagnation-day synthetic periods are not flagged (AT3).
  • Ground truth + match: planted BF-rate jumps + calendar intervals; lead time recovered on a planted lead (AT4).
  • Market study: synthetic returns with a planted CAR around events → recovered CAR/hit-rate/CI; planted-null returns → CAR ≈ 0 (AT5).
  • No-look-ahead guard: detection at t unchanged by appending future data; market alignment never precedes overpass+latency (AT6).
  • Graceful skip: a missing instrument is skipped with a note (AT7).
  • CLI: subcommands wired; existing commands unchanged (AT-REG-1).

All statistics tested on synthetic planted fixtures; no network in tests (market fetch mocked).

None (no UI). Static event-study plots eyeballed as outputs.

Risks and trade-offs

  • Multiple testing / overfitting (top risk). Many knobs (threshold, window, instrument, latency) → a positive CAR could be a search artefact. Mitigation: fix defaults before looking at market returns, record n_tests, adjust/flag multiplicity, lead with confirmed-event hypotheses (REQ-042, NFR-004).
  • Look-ahead leakage. Easy to accidentally use future data in baselines or trade before the overpass is knowable. Mitigation: causal baselines + explicit overpass+latency alignment + a guard test (REQ-041, AC-006).
  • Meteo false positives. Stagnation spikes mimic production events. Mitigation: the strengthened ventilation/stagnation control (REQ-005) and event-must-survive-meteo rule.
  • Cloud-gapped events. Real curtailments hidden by cloud become reported missed detections, lowering recall honestly — not silently absorbed (EDGE-005).
  • Few events on the partial series. Low statistical power; the full TROPOMI fetch (NOX-003.1 T10) materially helps. Mitigation: ERR-003 refuses an underpowered study; report power.
  • Ferrous-data reproducibility. No clean free SHFE/DCE API. Rejected: a paywalled dependency. Mitigation: lead with free miners+ETF; ferrous as documented snapshot or deferred (Q1).
  • Market edge realism. The operating rate is already published weekly; the edge is timeliness/gap-filling, not resolution. The study reports lead-vs-publication honestly; a null lead is a valid outcome.

Rollback plan

Additive: a self-contained new package + new CLI subcommands + new config; no existing behaviour changes. Rollback = revert the feature branch and delete the gitignored event/price artifacts. A problematic new dependency is removed and its subcommand disabled, leaving NOX-001/002/003 intact.
