A3ERA5 (Copernicus CDS) is the meteo normalisation sourcePending
Assumption
ERA5 10 m wind + boundary-layer height is sufficient for the meteo regress-out, ingested to a dated snapshot from the Copernicus CDS (reanalysis-era5-single-levels, server-side AOI subset). Exact variables/levels and regression form are open questions.
Reason
Developer-locked that meteo is in scope (2026-06-13); ERA5 is the standard public reanalysis used by Beirle v2 / Mao 2025. Source set to CDS (2026-06-13): ARCO chunk-1 downloads full global fields per timestep (~30–760 GB for the AOI series), so CDS server-side subsetting (~MB) is the right access pattern. Free CDS account credential read by name from .env; one-time licence acceptance required.
Risk level
Medium
Impact if wrong
If ERA5 access/latency or variable choice is wrong, meteo control is weakened; mitigated by snapshotting and configurable variable set.

A4Active-status filter on facilitiesPending
Assumption
Only facilities with status ∈ {operating, operating pre-retirement} define the footprint; retired/cancelled are excluded. The CSV is a static snapshot (status as-of compile date, not time-varying).
Reason
Retired/cancelled plants should not contribute current NOₓ; the CSV has no per-date status history.
Risk level
Medium
Impact if wrong
A plant retired mid-record (or restarted) would bias the footprint over part of the window; static status is a known limitation to note in the report.

A7A curtailment calendar is obtainable as a public seriesPending
Assumption
Tangshan production-curtailment periods can be assembled from public CREA/Mysteel-style reporting into an exogenous control series at weekly resolution.
Reason
Required to control the policy confounder (Li 2024); curtailments are widely reported for Tangshan.
Risk level
High — if no clean public calendar exists, the policy control is degraded to a coarse proxy.
Impact if wrong
Without it the indicator can invert (controls cut NOₓ while output rises). Mitigation: if unavailable, document the gap and report results as conditional; possibly use the CREA BF-starting-rate aux series as a proxy.
